
# StackCalc
### A simple calculator

StackCalc is a simple calculator that calculates basic 2D shapes' area and circumference,
and later, basic 3D shapes' volume and surface area

  
Supported shapes

- 2D

	- Circle

	- Rectangle

	- Triangle

- 3D

	- Cube

	- Sphere
